# Ryo's SBCL-Goodies

## About
This tap includes my port of [SBCL-Goodies](https://github.com/sionescu/sbcl-goodies/) to macOS arm64 platform. 

## How do I install these formulae?

`brew install li-yiyang/sbcl-goodies/<formula>`

Or `brew tap li-yiyang/sbcl-goodies` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "li-yiyang/sbcl-goodies"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
