# Li-yiyang Sbcl-goodies

## How do I install these formulae?

`brew install li-yiyang/sbcl-goodies/<formula>`

Or `brew tap li-yiyang/sbcl-goodies` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "li-yiyang/sbcl-goodies"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
